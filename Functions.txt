1. Создать функцию, которая принимает число и объект с тремя вариантами числительного. Возвращает одно из них в
соответствии с правилами русского языка.

2. Создать функцию, которая на вход принимает объект с тремя числительными и возвращает функцию. Возвращенная
функция должна принимать на вход число, и возвращать одно из них в соответствии с правилами русского языка.

3. Сделать генератор элемента, который будет принимать на вход родительский элемент, в котором будет создавать дочерний, и совйства в виде объекта.
Должны поддерживаться такие свойства:
tag - имя тега (по умолчаию div)
child - дочерний элемент, при указании помещается внутрь созданного элемента,
class - имя класса
id - айди элемента
place - место вставки элемента (одно из "beforebegin" - "afterend"), по умолчанию "beforeend"


пример вызова функции: elementHandler(root, {
  tag: "input",
  class: "name"
});
